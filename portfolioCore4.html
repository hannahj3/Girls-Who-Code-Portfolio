Core 4:

variable: storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.
function:A type of procedure or action that outputs a value?
conditional: features of a programming language, which perform different computations or actions depending on whether a programmer-specified boolean condition evaluates to true or false.
loops: a sequence of instruction s that is continually repeated until a certain condition is reached.
